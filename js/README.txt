js  文件夹分类说明：

===================================================

base:  基础依赖js文件，如 jquery.js

bootcomp:  bootstrap component， 单一的bootstrap的js插件，如模态框，提示框 等，如考虑页面加载速度及节省宽带流量 等可根据需要单一引入相应的js文件，详细说明见： http://v3.bootcss.com/javascript/#transitions

bootdist: bootstrap的完整版本，包含所有的js插件

vendor:  其他插件或者通用js, 如 underscore, bootstrap-datepicker 等

client:  真旅行前台客户端（用户使用端）

maintain:  真旅行后台维护端 

global： 真旅行公共js （前后端公用的模块）






说明：

以上为js文件分类说明，之后可根据需要添加新的目录，或删除多余的文件目录
